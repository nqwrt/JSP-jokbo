1.자바스크립트에서 아래를 예를 들어 설명하시오.
-- 함수 안에 함수(중첩 함수)
함수 내부에 만들어지는 함수로서 중첩은 한번만 가능.
언제사용? 
1) 함수 내부에서만 사용하는 기능일 때.
2) 중복 코드 또는 코드 그룹화.
~함수 내부의 큰 기능이나 중복 코드를 내부함수로 만들어 재사용 할 때
~중첩 함수는 자신을 포함한 함수의 지역변수에 접근 가능

-콜백함수
언제사용?
1) 주로 함수 내부의 처리 결과값을 함수 외부로 내보낼 때 사용(일종의 return 문)
2) 비동기 방식으로 처리할 때
장점으로는 로직 구현 부분과 로직 처리 부분을 분리 가능하다.

-클로저
변수는 클로저 현상에 의해 외부함수의 호출이 끝나더라도 사라지지않고 값을 유지하게된다.
장점으로는 연관 있는 변수와 기능을 하나의 함수로 묶어 독립적으로 실행 가능.
함수 내부에 데이터가 만들어지기 때문에 함수 외부에서는 수정할 수 없는 보호된 데이터를 
만들 수 있다.

-함수안에 또다른 함수
내부함수로 외부에서는 호출 불가.
